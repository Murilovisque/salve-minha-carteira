            Informar os dados da boleta
    Tela de visualização da quantidade das ações
        Lista das ação, com a quantidade total, menor/maior e valor médio
    Tela de detalhes da ação específica
        Lista de todas as compras das ações 
    Tela de planejamento
        Exibir dados de planejamento de compra de ações
            É uma compra de ações já comprada anteriormente?
                Sim. Deve exibir o menor valor, o maior e a média pagos por aquele daquela ação
            É uma venda de ações
                Deve indicar o valor que a ação deve estar para ser vendida com o percentual de lucro que o cliente deseja ter, dada a quantidade que irá vender                    
                    Venderá todas as ações daquele código?
                        Sim, então é só somar todas e acrescentar os 10%, depois dividir esse resultado com a quantidade de ações
                        Não, somar os valores das ações de menor para o maior valor e do maior para o menor valor, depois dividir cada resultado com a quantidade de ações. Afim de indicar as duas situações para venda das ações. Por enquanto não irá considerar vendas anteriores nesse calculo
    Entidades
        Acao
            Nome, Código
        Boleta
            Tipo (Compra/venda), Data, Valor, Quantidade, Ação
